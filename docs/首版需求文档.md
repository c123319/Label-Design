# 一、需求文档（Requirements Document）

## 1. 项目概述

开发一个 **批量生成证书/标签的在线工具**，用户可：

* 通过可视化编辑器设计模板
* 上传批量数据（Excel/CSV/JSON）
* 自动生成图片或 PDF
* 支持条码/二维码渲染
* 支持模板保存和多页管理

---

## 2. 功能需求

### 2.1 前端功能

1. **模板设计**

   * 拖拽文字、图片、条码、二维码、形状
   * 调整大小、旋转、颜色、字体、阴影、圆角
   * 元素层级控制（置顶/上移/下移/置底）
   * 支持占位符（如 {{product_name}}）

2. **文件操作**

   * 上传背景图片或 PDF
   * 导出 PNG / JPG / PDF
   * 保存模板 JSON 到后端

3. **批量数据操作**

   * 上传 CSV / Excel / JSON 文件
   * 数据预览和字段映射

4. **页面管理**

   * 多页模板设计
   * 页面复制 / 删除 / 切换

---

### 2.2 后端功能

1. **模板管理**

   * 保存模板 JSON
   * 查询、删除模板
   * 模板版本管理

2. **批量生成**

   * 读取模板 JSON
   * 使用上传数据替换占位符
   * 渲染条码 / 二维码
   * 导出图片 / PDF
   * 支持批量压缩打包下载

3. **接口设计**

   | 接口   | 方法   | 参数                  | 返回              |
   | ---- | ---- | ------------------- | --------------- |
   | 上传模板 | POST | JSON                | template_id     |
   | 上传数据 | POST | CSV/JSON            | job_id          |
   | 批量生成 | POST | template_id, job_id | ZIP 下载地址        |
   | 查询状态 | GET  | job_id              | progress / 完成结果 |

---

### 2.3 非功能需求

* **性能**：支持同时生成 1000+ 证书
* **安全**：文件上传需限制类型和大小
* **可扩展性**：模板类型和占位符可扩展
* **兼容性**：主流浏览器（Chrome/Edge/Firefox）

## 3. 批量生成流程

1. 用户上传模板 JSON → 后端保存
2. 用户上传批量数据 → 后端解析
3. 创建生成任务（Job）
4. 循环每行数据：

   * 替换占位符
   * 渲染文字/条码/二维码
   * 生成 PNG / JPG
5. 批量生成完成 → 压缩 ZIP 或生成 PDF
6. 返回下载链接

---

## 4. 技术选型

| 层    | 技术                                                      |
| ---- | ------------------------------------------------------- |
| 前端   | Vue3 / React, Fabric.js / Konva.js, AntDesign / Vuetify |
| 后端   | Node.js (Express/NestJS) 或 Python (FastAPI)             |
| 渲染   | Fabric.js + Node canvas / pdfkit / Pillow               |
| 数据存储 | 文件系统或对象存储（S3/COS）                                       |
| 批量任务 | Node.js Worker Threads 或 Python Celery                  |
| 压缩   | archiver (Node) / zipfile (Python)                      |

* 共享素材缓存（背景图、Logo）
* 限制每次批量生成数量或分页
* 支持异步任务和进度查询
