# 标签工作台 UI 规范标准文档

## 1. 文档说明

### 1.1 文档目的

本文档用于规范“标签工作台 / Label Studio”类标签设计编辑器的界面设计标准，确保产品在后续开发、迭代、组件扩展、页面新增过程中保持统一的视觉风格、交互体验和组件规范。

### 1.2 适用范围

本规范适用于以下模块：

* 顶部导航栏
* 顶部工具栏
* 左侧组件库
* 中央画布编辑区
* 标尺与网格系统
* 右侧属性面板
* 底部页面管理区
* 弹窗、抽屉、表单、按钮、图标、状态提示等通用组件

### 1.3 设计风格定位

整体风格采用：

* 极简
* 专业
* 工具型
* 蓝白科技感
* 高留白
* 低干扰
* 强操作可读性

产品定位为面向企业、制造业、打印场景的专业标签设计工具。

| 成功绿 | Success | #22C55E |
| 警告黄 | Warning | #F59E0B |

## 3.3 中性色

| 用途   | 颜色             | HEX     |
| ---- | -------------- | ------- |
| 一级文字 | Text Primary   | #1F2937 |
| 二级文字 | Text Secondary | #4B5563 |

| 禁用文字 | Disabled       | #C0C4CC |

## 3.4 颜色使用原则

* 蓝色只用于主操作、选中态、焦点态和关键状态。
* 大面积背景禁止使用高饱和蓝色。
* 编辑器区域应减少强色干扰，避免影响用户判断标签内容。
* 删除、错误、危险操作必须使用红色。
* 成功保存、任务完成使用绿色。
* 警告、未完成配置使用黄色或橙色。

# 7. 阴影规范

## 7.1 面板阴影

常规面板尽量不用重阴影，主要通过边框区分。

```css
box-shadow: 0 1px 2px rgba(16, 24, 40, 0.04);
```

## 7.2 画布阴影

标签画布需要轻微悬浮感：

```css
box-shadow: 0 8px 24px rgba(15, 23, 42, 0.08);
```

## 7.3 弹窗阴影

```css
box-shadow: 0 20px 40px rgba(15, 23, 42, 0.16);
```

# 19. 画布缩放规范

## 19.1 缩放控制

底部或右下角提供缩放组件：

```text
[手型拖拽] [-] [125%] [+] [适应窗口]
```

## 19.2 缩放范围

| 项    |    值 |
| ---- | ---: |
| 最小缩放 |  10% |
| 最大缩放 | 400% |
| 默认缩放 | 100% |
| 缩放步长 |  10% |

## 19.3 缩放原则

* 缩放时标尺、网格、元素必须同步变化。
* 元素真实尺寸不变，只改变视图比例。
* 标尺显示仍以 mm 为单位。
* 网格不可与标签画布错位。

# 25. 页面验收标准

## 25.1 视觉验收

* 页面整体以白色、浅灰、蓝色为主。
* 蓝色只用于选中、主操作和关键提示。
* 左侧、右侧、顶部、底部区域分区清晰。
* 所有图标风格统一。
* 字号、间距、圆角保持一致。
* 标签画布和工作区背景能明显区分。

## 25.2 交互验收

* 鼠标 Hover、Active、Disabled 状态完整。
* 元素选中态清晰。
* 缩放时网格、标尺、元素不发生错位。
* 右侧属性面板随元素类型动态切换。
* 底部页面缩略图可正确选中、添加、删除。
* 保存、导出、打印按钮状态明确。

## 25.3 工程验收

* 使用统一 CSS 变量。
* 禁止页面中散落硬编码颜色。
* 组件间距基于 4px 栅格。
* 主要组件需可复用。
* UI 组件需支持主题扩展。
* 画布相关计算需统一坐标系。

# 27. 总结

本 UI 规范以“专业标签设计编辑器”为核心定位，采用蓝白极简工具型风格。设计重点是：

* 保持清晰的信息分区；
* 降低视觉干扰；
* 强化编辑器操作效率；
* 保证画布、标尺、网格、属性面板的一致性；
* 支持后续模板、数据绑定、批量生成、打印导出等业务扩展。

所有新增页面和组件均应遵循本文档中的色彩、字体、间距、圆角、按钮、面板、画布和交互状态规范。
