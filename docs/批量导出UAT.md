# 批量导出 UAT 清单

用于验证「数据预览 → 批量 ZIP 导出」与阶段 A 验收标准。

## 测试数据

样例文件：[`samples/batch-export-sample.csv`](samples/batch-export-sample.csv)

| 字段 | 说明 |
| ---- | ---- |
| 产品名称 | 中文文本占位符 |
| 规格型号 | 条码 / 二维码内容 |
| 序列号 | 文件命名模板 `{{序列号}}_标签` |
| 生产日期 | 日期文本 |
| 编号 | 纯数字 |

## 模板准备

1. 新建标签模板（建议 100×70mm）
2. 添加文本：`产品名称：{{产品名称}}`，`规格：{{规格型号}}`，`日期：{{生产日期}}`
3. 添加条码，内容绑定 `{{规格型号}}`
4. 添加二维码，内容绑定 `{{序列号}}`
5. （可选）复制为第 2 页，用于多页导出测试

## 用例

### UC-01 数据上传与预览

| 步骤 | 操作 | 预期 |
| ---- | ---- | ---- |
| 1 | 上传 `batch-export-sample.csv` | 解析成功，显示 5 条数据 |
| 2 | 进入预览，切换第 1～5 条 | 文本、条码、二维码均替换正确 |
| 3 | 预览第 1 条 | 产品名称显示「伺服电机」，条码内容为 MP-SM1000 |

### UC-02 单页 ZIP 批量导出

| 步骤 | 操作 | 预期 |
| ---- | ---- | ---- |
| 1 | 批量生成 → 校验 | 校验通过 |
| 2 | 文件命名留空 → 下载 ZIP | 下载单个 `.zip` |
| 3 | 解压 | 内含 5 张 PNG：`模板名_001.png` … `_005.png` |
| 4 | 打开 `_001.png` | 与预览第 1 条视觉一致 |

### UC-03 字段文件名

| 步骤 | 操作 | 预期 |
| ---- | ---- | ---- |
| 1 | 文件命名填 `{{序列号}}_标签` | — |
| 2 | 下载 ZIP 并解压 | 文件名为 `SN001_标签.png`、`SN002_标签.png` … |

### UC-04 多页导出

| 步骤 | 操作 | 预期 |
| ---- | ---- | ---- |
| 1 | 模板含 2 页，上传 5 行数据 | 弹窗提示「5 条，2 页模板」 |
| 2 | 下载 ZIP | 共 10 张 PNG |
| 3 | 检查命名 | `001_1.png`、`001_2.png` … `005_2.png` |
| 4 | 每对页面 | 同一行数据，两页占位符替换一致 |

### UC-05 进度与取消

| 步骤 | 操作 | 预期 |
| ---- | ---- | ---- |
| 1 | 批量导出进行中 | 显示「正在生成第 N / M 张」 |
| 2 | 点击「停止导出」 | 不下载 ZIP，画布恢复，提示已取消 |
| 3 | 关闭弹窗（导出中） | 同停止导出 |

### UC-06 校验拦截

| 步骤 | 操作 | 预期 |
| ---- | ---- | ---- |
| 1 | 模板含 `{{不存在字段}}` | 校验报错，无法导出 |
| 2 | 条码字段为空（某行） | 校验警告，仍可导出 |

## 阶段 A 验收对照

- [ ] 32 行 Excel → 预览正确 → ZIP 内 32 张，占位符均已替换（可用样例扩展行数验证）
- [ ] 2 页模板 → 每行 2 张图
- [ ] 中文占位符 `{{产品名称}}` 批量替换正确
- [ ] 条码 / 二维码导出与预览一致
